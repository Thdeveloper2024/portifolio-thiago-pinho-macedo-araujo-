# 🌍 Sistema de Análise de Microclima
 
## 📝 Descrição do Projeto
Este projeto consiste em um sistema de análise de microclima desenvolvido em Python, que avalia diferentes locais com base em três fatores principais: temperatura, qualidade do ar e umidade.

O objetivo principal é simular um sistema inteligente de avaliação ambiental, atribuindo notas para cada variável e gerando uma média final que representa a qualidade do ambiente analisado.

O sistema percorre uma lista de locais previamente definidos, aplica regras condicionais para cada indicador climático e exibe um relatório completo com as notas individuais e a nota final de cada local.
 
## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3  
* **Bibliotecas:** Nativas do Python (sem dependências externas)  
* **Ferramentas:** Terminal / Prompt de Comando  
 
## 📊 Resultados e Aprendizados
O projeto demonstrou eficiência na análise de múltiplos fatores ambientais, sendo ideal para desenvolvimento de lógica mais avançada.

* **Avaliação Multicritério:** Implementação de cálculo baseado em múltiplas variáveis (temperatura, ar e umidade).
* **Uso de Estruturas Modernas:** Aplicação de `match case` para controle de fluxo.
* **Classificação Inteligente:** Definição de notas com base em intervalos específicos.
* **Cálculo de Média Final:** Geração de uma nota geral para tomada de decisão.
 
## 🔧 Como Executar
1. Clone ou copie o código para um arquivo `.py`.
2. Certifique-se de ter o Python instalado.
3. Execute o comando: `python main.py`.
