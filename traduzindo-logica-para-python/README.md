# 🧾 Sistema de Processamento de Vendas e Análise Climática

## 📝 Descrição do Projeto

Este projeto consiste em um sistema simples desenvolvido em Python que realiza duas operações principais: processamento de vendas e análise de dados climáticos.

O objetivo principal é aplicar conceitos fundamentais de programação, como estruturas condicionais, laços de repetição e manipulação de dados, simulando cenários reais de negócio e monitoramento ambiental.

Na parte de vendas, o sistema permite registrar produtos, calcular o valor total da compra e aplicar descontos automáticos com base em regras definidas. Já na análise climática, o sistema coleta temperaturas ao longo de uma semana, calcula a média e identifica condições extremas que exigem atenção.

## 📊 Resultados e Aprendizados

O projeto demonstrou eficiência na execução das regras propostas, sendo ideal para consolidação de lógica de programação.

- Automação de Cálculos: Implementação de cálculo automático de totais e descontos em compras.  
- Validação de Dados: Aplicação de regras para evitar entradas inválidas (preço e quantidade).  
- Análise de Dados Simples: Cálculo de média e identificação de padrões climáticos.  
- Detecção de Condições Críticas: Identificação de temperaturas extremas com geração de alertas.
